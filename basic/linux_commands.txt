ls - просмотр содержимого текущей директории
ls путь_до_директории - просмотр содержимого указанной директории
ls -a - просмотр скрытых файлов директории
mkdir название_директории - создание директории
cd путь_до_директории - мсена директории
"." - ссылка на текущую директорию
".."- ссылка на директорию выше/родительскую
"~" - ссылка на доиашнюю директорию
pwd - вывод абсолютного пути в консоль

абсолютный путь - это путь начинающийся с домашней lbhtrnjhbbf,cjk.nysq genm - 'nj genm yfxbyf.obqcz c ljvfiytq директории(рщьу/вщ(home/donny/some_dir)

rmdir - удаление пустой директории
rm -rf - удаление директории с содержимым

touch - создание файлов в текущей директории
rm - удаление файлов
rm -rf имя_файла - принудительное удаление файлов

cat имя_файла - вывод содержимого файла в терминал
cat имя_файла >> имя_другого_файла - дозапись содержимого одного файла в другой

echo "какойто текст"- вывод указанного текста в терминал
echo "какойто тест" >> название_файла - дозапись указанного текста в файл
